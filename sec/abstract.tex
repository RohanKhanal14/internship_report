\section*{\Large\centering \textbf{ABSTRACT}}
\addcontentsline{toc}{section}{Abstract}

This report presents the work carried out during the internship as a \textbf{News Portal Administrator} at \textbf{Kendrabindu Online Pvt. Ltd.}. The internship was conducted with the objective of gaining practical exposure to web-based content management systems, database-driven applications, website administration, security implementation, and performance optimization in a real-world media environment.

During the internship period, various technical and administrative activities were performed to manage and enhance the organization's online news portal. Major tasks included system analysis of the Content Management System (CMS), database management using MySQL, implementation of role-based access control, website performance optimization, SEO improvements, server monitoring, API integration for social media automation, and basic web security practices such as input validation and protection against common vulnerabilities.

Additionally, activities related to hosting configuration, SSL implementation, backup automation, website analytics, system testing, and documentation were carried out. The internship also involved maintaining data integrity, improving user engagement through structured content categorization, and enhancing system reliability through routine maintenance and debugging.

Overall, the internship provided practical knowledge of web technologies, database management systems, server administration, and system optimization practices. It successfully bridged the gap between academic coursework in BSc CSIT and real-world industry application in the digital media sector.
